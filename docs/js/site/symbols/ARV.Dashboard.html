<static> ARV.Dashboard.addTextComponents(componentClass, type, oldComponent, position)
Adds the text component i.e. heading, label or a description field
Parameters:

{string} The class of the component which can be label, heading or description
type
{string} type of the component which can be textbox or text area
oldComponent Optional
{Object} Optional field if given it means that the the new component to be added will be the clone of the oldComponent
position Optional
{Object} The position of the oldComponent. So the new component to be added will be added in the same position
